## 🚨 إصلاحات حرجة (أولوية قصوى)

### 1. أمان — CORS مفتوح للجميع
- **الحالة**: `allowed_origins: ['*']` — أي موقع يستطيع إرسال طلبات للـ API
- **الحل**: تقييد الـ origins لـ `erpp.tech` فقط
- **التأثير**: 🔴 حرج

### 2. أمان — التوكنات لا تنتهي صلاحيتها
- **الحالة**: `expiration: null` في إعدادات Sanctum
- **الحل**: ضبط `expiration` لمدة معقولة (مثلاً 24 ساعة) + refresh token
- **التأثير**: 🔴 حرج

### 3. أمان — لا يوجد Rate Limiting
- **الحالة**: لا حماية من الطلبات المكثفة (brute force)
- **الحل**: إضافة `throttle` middleware على routes الـ API
- **التأثير**: 🔴 حرج

### 4. باج — حذف ملفات المشاريع
- **الحالة**: `ProjectController.deleteFile()` يحذف السجل فقط دون حذف الملف الفعلي
- **الحل**: إضافة `Storage::disk('public')->delete($file->file_path)` قبل حذف السجل
- **التأثير**: 🟡 متوسط (تراكم ملفات على السيرفر)

### 5. صلاحيات — ProjectController + UserController
- **الحالة**: لا يوجد authorization checks في بعض العمليات
- **الحل**: إضافة Policy checks لجميع العمليات
- **التأثير**: 🟡 متوسط

- 🟡 لا يوجد CI/CD = deploy يدوي ممكن يسبب مشاكل
- 🟡 لا يوجد monitoring = لو حصل مشكلة مش هنعرف بسرعة
